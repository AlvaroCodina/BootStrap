Documentación

Estructura

En todo el Sitio Web uso BOOTSTRAP para hacer la estructura y el diseño básico. La mitad de los imports para que funcione BootStrap son CDN y los mas pesados están en local para que tarde menos en cargarlos.

Fuentes

Uso una fuente de Google Fonts 'Poiret One' para toda la WEB, para cada etiqueta (h1, h2, p) le pongo un tamaño según con que dispositivo se use

Paleta de colores

Usé la pagina de paletton para escoger el color de fondo y de las letras.

Multimedia

Imágenes: Uso de Sprites para ahorrar carga y responsivas, también usé un carrusel de BootStrap

Vídeo: Responsivo también y de YouTube para que cargue más rápido y ocupe menos
